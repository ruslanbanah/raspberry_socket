# RaspberryPI control via SocketIO

### Getting Started
1. Set up config file: _config.json_
2. Install dependency
* ``` 
    yarn install 
  ```
3. Start server
* ```
    yarn start
  ```

### List edvices :
```
bash ls /dev | grep ttyACM
```
